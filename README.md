# BLACTIFY

A premium e-commerce experience for timeless essentials.

## Features
- **Admin Dashboard**: Comprehensive management of products, categories, and orders.
- **Product Gallery**: High-performance image loading and sleek UI.
- **Secure Checkout**: Integrated with Razorpay for seamless payments.
- **Order Tracking**: Real-time status updates for customers.
- **PWA Ready**: Installable application on iOS and Android.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS & Framer Motion
- **Payments**: Razorpay
- **Email**: Resend
- **Auth**: Firebase Authentication

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Setup**:
   Copy `.env.local.example` to `.env.local` and fill in the required keys.

3. **Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
